 # 📝 React To-Do List Application

A responsive To-Do List Application built using React.js and Vite.

## 🚀 Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Mark tasks as Completed/Pending
- Search tasks
- Filter Completed and Pending tasks
- Task Categories
- Priority Levels
- Due Date
- Local Storage (Data persists after refresh)
- Task Summary Dashboard
- Responsive UI

## 🛠️ Technologies Used

- React.js
- JavaScript
- CSS3
- Vite
- Local Storage
